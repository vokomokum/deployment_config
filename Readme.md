Vokomokum Foodsoft Deployment (based on Foodsoft.net)
========================

This repository contains information and tools for the management of [app.foodcoops.net](https://app.foodcoops.net) NEEDS UPDATE. This is a fork of the archived public repository [https://github.com/foodcoops/foodcoops.net](https://github.com/foodcoops/foodcoops.net). 

Since November 2025 our deployment is [Ansible](https://www.ansible.com) based. Have a look at the corresponding folder for further information and instructions.

In the *documentation* directory you will find several administration guides.
